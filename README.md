# Progetto per il triage automatico dei ticket con ML 

## pw18-ticketing

### Prototipo1

Contiene il codice relativo alla Fase 1 del PW.

Si può eseguire `prototipo.py` tramite IDLE Python.

### Prototipo2

Contiene il codice relativo alla Fase 2 del PW.

Si può eseguire il codice tramite Colab Notebook.

è sufficiente eseguire il file `dashboard.ipynb` per vederne il funzionamento

### Prototipo3

Contiene il codice relativo alla Fase 3 del PW.

Si può eseguire il codice con Colab Notebook.

Dentro la cartella notebooks ci sono le 3 versioni descritte nel report.

### Export

Contiene i seguenti file:
1. ticket_input.csv: csv con una decina di ticket da fare valutare ai prototipi
2. export_validation_poc2.ipynb: codice per la predizione dei ticket usando il PoC2
3. predizioni_ticket_poc2.csv: csv di output generato automaticamente
4. predizioni_manuali_ticket_poc3.csv: raccolta manuale delle predizioni dei ticket usando la dashboard del PoC3
